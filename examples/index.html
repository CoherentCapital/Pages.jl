Test PagesJL as website server!
This is similar to (basically a copy) the plotly.jl example but modified to demonstrate that pages and documents can effectively be fetched from within a "folder" the drawback to the method I am employing is that instead of just writing:
"<link rel=\"stylesheet\" href=\"/css/Styles.css\" />"

You have to specify the folder as well. Like this:
"<link rel=\"stylesheet\" href=\"/MyCoolSite/css/Styles.css\" />"
It may be possible to fix this and conform more closely to standards by inserting the folder name automattically before each request is sent. I'm open to suggestions!
